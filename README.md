# Transformation Control Tower Dashboard

This project is a self-initiated simulation of a Transformation Control Tower dashboard.

It aims to replicate how strategic initiatives can be monitored at Group level through:
- Cross-functional KPIs (Finance, HR, Operations)
- Programme tracking (on-track / at-risk / delayed)
- Savings and productivity monitoring
- Executive-level summaries and action tracking

Built to better understand the role of a Control Tower Analyst.

## Tech Stack
- HTML5
- CSS3 (with CSS variables & animations)
- JavaScript (vanilla)
